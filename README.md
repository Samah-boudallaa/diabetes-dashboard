# Diabetes Analytics Dashboard
## Project Overview

The Diabetes Analytics Dashboard is an interactive data visualization application built with Streamlit and Plotly.

The objective of this project is to explore and analyze the key factors influencing diabetes using the Pima Indians Diabetes Dataset.

## This project focuses on:

Exploratory Data Analysis (EDA)

Interactive filtering

KPI visualization

Correlation analysis

Insight generation

### Dataset Information

Source: Kaggle
Dataset: Pima Indians Diabetes Dataset

## The dataset contains medical diagnostic measurements for female patients, including:

Pregnancies: Number of pregnancies

Glucose: Plasma glucose concentration

BloodPressure: Diastolic blood pressure

SkinThickness: Triceps skin fold thickness

Insulin: Serum insulin level

BMI: Body Mass Index

DiabetesPedigreeFunction: Genetic influence

Age: Patient age

Outcome: 0 (Non-Diabetic), 1 (Diabetic)

## Project Objectives

Identify the most influential factors related to diabetes.

Provide interactive tools for data exploration.

Build a clean and professional analytics dashboard.

## Key Insights

Glucose level shows strong correlation with diabetes outcome.

Higher BMI values are associated with increased diabetes risk.

Diabetes prevalence increases with age.
